With this hands-on workshop, we will explore the potential of variable fonts on the web. We will cover how you might take advantage of the relatively new font format, from fine-tuning typography to animation. Participants will have access to a template for a single-serve website, along with variable typefaces for testing. Familiarity with HTML and CSS is helpful but not required; a willingness to engage with code is required.
