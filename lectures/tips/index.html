Tips and Tools

Considerations in your type design process

1. Print & Proof at Size

Don’t be shy with serifs
Small details can get lost—make sure your design decisions feel intentional

2. Draw letters in context

Evaluate them in spacing strings (make a shortcut) and
paragraphs—Word-o-mat it!

3. Expand your Edit View

The default width of the edit view isn’t wide enough for seeing glyphs well in paragraphs.
Adjust Glyphs settings / preferences > Appearance > Text View Width to at least 24000.

4. Nail down your control characters

Refine your “phones RHODES” and their spacing first,
then proceed to other letters

5. Setup Sample Strings

Under Glyphs > Settings > Sample Strings, setup these spacing strings. You can then access them anytime through Edit > Select Sample Text (⌥ + ⌘F)

6. White shapes & black shapes

Pay close attention to shapes being formed by both the outside and interior outlines —
consider how the letter contours are carving out spaces.

7. Testing fonts

Don’t install fonts onto your computer—use the Adobe Fonts folder, and make sure it’s at the root level instead of the user level. /Library/Application Support/Adobe/

If you haven’t done so yet, you can download a sample Proof template from the shared Drive—make sure you adapt the font sizes to your intended point size.

8. Consult Glyphs Forum

The Glyphs Forum has a wealth of information, and searching it is often helpful.
The creators are very engaged with responding to inquiries.

Keyboard shortcuts

Description Shortcut
Send to background ⌘A (select all) + ⌘J
Toggle between background and foregraound ⌘B
Swap background and foregraound ⌃ + ⌘J
Move by 10 units ⇧ + arrow keys
Adjust sidebearings, left side (by ten units) ⌃ + arrow keys ( + ⇧ )
Adjust sidebearings, right side (by ten units) ⌘ + arrow keys ( + ⇧ )
Kern (by ten units) ⌥ + ⌘ + arrow keys ( + ⇧ )
Align selection ⇧ + ⌘A
Add Extremes points* ⌘ + click on path
Toggle measurement line ⌃ + ⌥ + ⌘ + drag
Correct Path direction (in all masters) ⇧ + ⌘R ( + ⌥ )
Move only selected on-curve point (by ten units) ⌥ ( + ⇧ )+ arrow keys
Move selected off-curve point symmetrically (by ten units) ⌃ + ⌥ ( + ⇧ ) + arrow keys

*Extreme points are on-curve nodes that lie at 90°. They help structure the curves cleanly. By default, Glyphs doesn’t support adding extremes only to a selected segment via keyboard, but you can do this via a script.

Generally speaking, keeping the shift pressed adjusts arrow-key increments to 10 units; shift + command adjusts them to 100 units.

The selected letter is the glyph on the right side of the cursor. You can adjust both the left and right side bearings of the glyph in this way, using those different keyboard shortcuts above.

Plugins

  • Angled Handles
  • All Angles (custom plugin at Occupant Fonts)

  • Stem Thickness

Reporter plugins

The Reporter plugin will help you manage your installed plugins in your right sidebar

Word-o-mat!

Once installed, select the glyphs you want to test (if not running it on all characters,) and activate it through Edit > word-o-mat. If you run into trouble with the installation, check out this post on the forum.
